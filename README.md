# ProGlycProt-v3
“Comprehensive database of experimentally validated prokaryotic glycoproteins, glycosyltransferases, and accessory enzymes (Version 3.0)”

**ProGlycProt** is a comprehensive database of experimentally validated **prokaryotic glycoproteins, glycosyltransferases, and accessory enzymes**. This repository supports research in microbial glycobiology, virulence, and biotechnology.

## Key Features
- **Database Size:** 1,655 entries (1,314 glycoproteins, 152 glycosyltransferases, 189 accessory proteins)  
- **3D Structures:** AlphaFold2 models for 1,044 glycoproteins and 113 glycosyltransferases  
- **Glycosylation Types:** N-, O-, and S-linked glycosylation  
- **Patent Integration:** 51 patents related to glycosylation technologies and enzymes  
- **Data Sources:** PRIDE datasets, literature curation, NCBI, UniProt, EMBL  

## Highlights
- Newly discovered glycoproteins (45 N-glycoproteins) from in-silico analysis of PRIDE datasets  
- Phylum-wise distribution and domain-level classification (archaea and bacteria)  
- User-friendly web interface and search functionalities: [ProGlycProt](https://www.proglycprot.org)  
- Updated bibliographic references and hierarchical organism classification
